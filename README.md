Welcome to the **Programming Concepts** repository. This repository contains the complete suite of programs and Jupyter Notebooks conducted for the "Programming Concepts" subject as part of the Post Graduate Certificate Programme in IT Infrastructure and Systems Security (PGCP ITISS) at CDAC-ACTS, Pune, in April 2026.

## 📌 Repository Overview

Designed to facilitate hands-on learning, this curriculum strictly adheres to a 90% practical structure. The materials guide learners through foundational to advanced Python concepts, encompassing core data structures, system operations, object-oriented paradigms, and network programming.

## 🗂️ File Structure & Curriculum

The coursework is organized into the following sequential notebooks:

* **`01 Python Basics.ipynb`**: Introduction to Python syntax, variables, and basic operators.
* **`02 Decision Making.ipynb`**: Conditional statements (`if`, `elif`, `else`).
* **`03 Loop Statements.ipynb`**: Iterative structures and control flow (`for`, `while`).
* **`04 Lists.ipynb`**: List creation, manipulation, and comprehension.
* **`05 Tuples.ipynb`**: Immutable sequence types and tuple operations.
* **`06 Dictionary.ipynb`**: Key-value pair mapping and dictionary methods.
* **`07 Set.ipynb`**: Unordered collections and mathematical set operations.
* **`08 Functions.ipynb`**: Defining functions, handling parameters, and return values.
* **`09 Modules.ipynb`**: Creating, importing, and utilizing modules.
* **`10 File Handling.ipynb`**: Reading from and writing to files.
* **`11 Object Oriented Programming.ipynb`**: Classes, objects, inheritance, and encapsulation.
* **`12 OS Module.ipynb`**: Interacting directly with the operating system.
* **`13 Regular Expressions.ipynb`**: Advanced pattern matching and string manipulation.
* **`14 Server Program.ipynb` & `15 Client Program.ipynb`**: TCP socket programming for client-server network communication.
* **`16 UDP Sender.ipynb` & `17 UDP Receiver.ipynb`**: Network programming using the User Datagram Protocol (UDP).
* **`18 Python Libraries.ipynb`**: Practical implementation and overview of essential Python libraries.
